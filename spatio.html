Basic models

$$ S: \text{Susceptible}, I: \text{Infective}, R: \text{Recovered} $$ $$ \begin{aligned} \frac{dS(t)}{dt} &= -\frac{\beta}{N}S(t)I(t)\\ \frac{dI(t)}{dt} &= \frac{\beta}{N}S(t)I(t) - \gamma I(t)\\ \frac{dR(t)}{dt} &= \gamma I(t) \end{aligned} $$ This is known as the Deterministic SIR model.
Estimation? Inference?

SIR Model

Rate of infection, $\lambda(t) = \frac{\beta}{N}S(t)I(t)$
Rate of recovery $= \gamma I(t)$

This motivates the Stochastic SIR model.

Stochastic SIR Model

We shall form a Continuous Time Markov Chain.
From any state $\left( S(t), I(t)\right)$, two types of jumps are possible.
Event Rate
$\left( S(t), I(t) \right) \rightarrow \left( S(t)-1, I(t)+1 \right)$ $\frac{\beta}{N}S(t)I(t)$
$\left( S(t), I(t) \right) \rightarrow \left( S(t), I(t)-1 \right)$ $\gamma I(t)$

Spatial extension:
Metapopulation model

Divide the population into $K$ subpopulations. Assume homogeneous mixing within each subpopulation.
Here is the deterministic SIR model: For $1\le k \le K$, $$ \begin{aligned} \frac{dS_k(t)}{dt} &= -\frac{\beta_k}{N_k}S_k(t)\left(\sum_{l=1}^{K} w_{kl}I_l(t) \right)\\ \frac{dI_k(t)}{dt} &= \frac{\beta_k}{N_k}S_k(t)\left(\sum_{l=1}^{K} w_{kl}I_l(t) \right) - \gamma I_k(t)\\ \frac{dR_k(t)}{dt} &= \gamma I_k(t) \end{aligned} $$

Stochastic Metapopulation model

For the deterministic model, the rate is now, $$ \lambda_k(t) = \frac{\beta_k}{N_k}S_k(t)\left(\sum_{l=1}^{K} w_{kl}I_l(t) \right) $$
Thus we can think of a CTMC where from the state $\left( S_1(t), \ldots, S_K(t), I_1(t), \ldots, I_K(t) \right)$, it can jump to one of $2K$ many states:
  • One susceptible in $k^\text{th}$ group gets infected. Rate: $\lambda_k(t)$
  • One infected in $k^\text{th}$ group recovers. Rate: $\gamma I_k(t)$

Parameter Estimation

Suppose we have observed the process for time $(0, T]$. We have individual level data: $$ \left\{ (t_i, y_i, k_i): i=1, \ldots, n \right\} $$
$n$ : Number of infections
$t_i$ : Time of infection for $i^\text{th}$ infected
$y_i$ : Length of infectious period
$k_i$ : Corresponding subpopulation

Parameter Estimation

Let the PDF of infectious period be $f_{I\rightarrow R}$. Usually, exponential with rate $\gamma$.
Then the log-likelihood is given by, $$ l\left( \boldsymbol{\beta}, \gamma \right) = \sum_{i=1}^{n}\log f_{I\rightarrow R}(y_i) + \sum_{i=1}^{n} \log \lambda_{k_i}(t_i) - \int_0^T \sum_{k=1}^K \lambda_k(u) du $$ Involves integration!! But $\lambda_k$'s are piecewise constant. So it is tractable.

Discrete Time Model

Consider small time intervals: $[t, t+\delta t)$. Assuming constant intensity $\beta I(t)$ over the interval, the probability that a susceptible would escape infection in that interval is given by $\exp \left( -\beta I(t)\cdot\delta t \right)$
$$ C: \text{Number of new infections}\\ D: \text{Number of new recoveries} $$ $$ \begin{aligned} C_{[t, t+\delta t)}& \sim \text{Bin}\left( S(t), 1 - \exp \left( -\beta I(t)\cdot\delta t \right) \right)\\ D_{[t, t+\delta t)}&\sim \text{Bin}\left( I(t), 1-\exp\left( -\gamma\cdot\delta t \right) \right) \end{aligned} $$

Discrete Time Model

$$ \begin{aligned} S_t &= S_{t-1} - C_t\\ I_t &= I_{t-1} - D_t + C_t \end{aligned} $$ Poisson approximation gives, $$ \begin{aligned} C_t &\sim \text{Poi}\left( \beta S_{t-1}I_{t-1}\cdot\delta t \right)\\ D_t &\sim \text{Poi}\left( \gamma I_{t-1}\cdot\delta t \right) \end{aligned} $$
So a Poisson GLM can be fitted to estimate the parameters, using some transformation of past counts as regressors.

Discrete Time Model

So a Poisson GLM can be fitted to estimate the parameters, using some transformation of past counts as regressors.

In case of multiple subpopulations, we can model $C_{t, k}, D_{t, k}$ similarly. We have to replace $\beta I(t)$ with some function of $\lambda_k(t)$.

Problems

  • Information on number of susceptibles is usually not available.
  • There might be seasonal dependence. Some function of $t$ can be a useful regressor.
  • Some other covariates can also play a role.

To address these issues, Leonhard Held, Michael Höhle and Mathias Hofmann proposed an endemic-epidemic model, which is now known as the HHH model.

Endemic-epidemic model

Motivation

Branching process with immigration: $$ y_t|y_{t-1} \sim \text{Poi}(\mu_t)\\ \mu_t = \nu + \lambda y_{t-1} $$
  • Without the endemic part, this is an approxiation to a chain binomial model.
  • The endemic part captures the temporal pattern when $\nu$ is replaced by some $\nu_t$.

Cart-Pole problem

Classical problem of balancing a pole by applying force on a cart.
We have a system like this:

Want to balance the stick

Cart-Pole problem

Here are the values: $$M=1kg, m=0.1kg, L=1m,\\ g=9.8ms^{-2}, \Delta t=0.02s$$
After each 0.02s, the decision maker decides to apply a force $F$ on the cart for the next 0.02s. Possible values of $F$ are $-10N, 0N, 10N$
Objectives are:
  • To keep the cart within boundary.
  • To keep the angle of the pole withinn a range ($-12^o$ to $12^o$)
  • To keep the system within these limits upto $200^{th}$ timepoint.

Cart-Pole problem

The dynamics of the system is described by the following:
Define $$ p(t)=\frac{F+m\omega^2(t)L \sin(\theta(t))}{M+m} $$ Then, $$ \frac{d\omega(t)}{dt}=\frac{g\sin(\theta(t))-p(t)\cos(\theta(t))}{L\left(\frac{4}{3}-\frac{m\cos^2(\theta(t))}{M+m}\right)}\\ \frac{dv(t)}{dt}=p(t)-\frac{mL\frac{d\omega(t)}{dt}\cos(\theta(t))}{M+m} $$

Cart-Pole problem

Given $x, v, \theta, \omega$ at time $t$, we can compute the state for time $t+\Delta t$. And apply the force accordingly.
Yes, the process is deterministic if the precise measurements are given. But a little child can also balance a pole on his/her hand, without knowing the physics or the precise measurements.
Here we try to mimic the same behaviour. We shall provide only some discrete info about the state and see if we can find a good policy.
Drawback: Markov property may no longer be valid. But let’s see how it goes.
